## Tech Stack

**Client:** React, React-bootstrap, css

**Server:** Node, Express

**Database:** MongoDB


## API Reference


#### Add member

```http
  POST /v1/members/
```

#### Edit member

```http
  PUT /v1/members/${id}
```

#### Get all members

```http
  GET /v1/members/
```


#### Get member

```http
  GET /v1/members/${id}
```


#### Get all countries

```http
  GET /v1/countries/
```




## Installation

1.Open the project using code editor or command prompt


2.Install node package using npm both client and backend

```bash
Backend:
  npm install

Client:
  cd Client
  npm install
```
